**AI for Society – Creative Frontend Showcase**

AI for Society is a visually creative and interactive frontend web application developed to explore the capabilities of Google AI Studio in generating stylish user interfaces and animated layouts. The project focuses on presenting AI-related topics in an aesthetic and engaging way using modern frontend design concepts such as smooth transitions, card-based navigation, and dark-themed visual styling.

This project was created mainly to understand how Google AI Studio can assist in building creative and animated frontend applications. The emphasis of this project is on user interface design and visual presentation rather than on implementing complex AI logic or backend systems.

**Project Overview**

The main objective of this project is to build a creative home screen and topic-based interface where users can interact with animated content cards related to AI for Society themes. When a topic is selected, five new cards appear related to that topic. Only the selected card remains centered while the other cards smoothly move out of the screen to the left and right sides. This animation provides a clean and modern user experience.

The home screen includes a large and stylish title displaying “MindaMatrix” along with the subtitle “Android App Development with Generative AI,” creating a strong visual identity for the internship project. The entire application uses a dark and shining aesthetic background with modern visual effects to give a futuristic and professional look.

This project was designed as a learning exercise to explore creative frontend development using AI-generated guidance and prompts from Google AI Studio.

**Technology Stack**

This project is built using React with Vite as the build tool. Animations and transitions are implemented using modern animation libraries such as Framer Motion and GSAP. Styling is done using custom CSS with a focus on dark-themed and aesthetic design. Git and GitHub are used for version control and project hosting.

**AI Studio Prompt Used**

The project was generated using a structured prompt in Google AI Studio that described the complete requirements of a creative and animated frontend web application focused on AI for Society themes. The prompt requested a dark-themed and aesthetic user interface with a modern home screen design that displays the title “MindaMatrix” along with the subtitle “Android App Development with Generative AI.”

The prompt specified that when a user selects a topic, five new related cards should appear on the screen. Only the selected card should remain visible in the center, while the remaining cards should smoothly animate and move out of the screen toward the left and right sides. The background was required to be dark, stylish, and visually appealing with a futuristic aesthetic feel.

The prompt also included requirements for smooth animations, responsive layout, clean component structure, and visually engaging transitions. It instructed the system to focus mainly on creative frontend design and user experience rather than on implementing complex AI logic or backend functionality.

The complete project structure, UI layout, and animation behavior were generated using Google AI Studio and later reviewed, refined, and finalized by the developer.

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
